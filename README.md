Project Overview

This project is a 5-page website for a streetwear brand called "Cozziano". The purpose of the website is to show the brand, display products, and allow customers to get in touch with the business.

Website Goals

The main goals of the website are to show information about the brand, allow customers to view clothing products, give users a way to send enquiries, and provide contact details for the business.

Website Features

The website includes a navigation menu that appears on all pages. It has a homepage, an about page, a products page, an enquiry form, and a contact page.

Colours Used

The website uses a black background with white text. Red is used for highlights and links. The footer is dark to match the overall theme.

Fonts Used

Arial is used for the main text and Georgia is used for headings.

How I Built It

I built the website using Visual Studio Code. I used HTML to create the structure and CSS to style the pages.

Questions I Asked While Building

While building the website, I asked questions like how to create a navigation menu, how to style the website using CSS, how to create a form, and how to link multiple pages together.
